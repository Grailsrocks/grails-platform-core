APIs and Tags for advanced Plugin integration

(Quick Reference)

s:createLogoutLink

Purpose

Returns a URL that will logout the use when visited.

This gets the URL from the security provider that is installed.

To render a button that will take the user to this page, see s:logoutButton.

Example

<s:createLogoutLink/>

Attributes

None.
