APIs and Tags for advanced Plugin integration

(Quick Reference)

p:dummyText

Purpose

Generates dummy text to use in work-in-progress UIs.

Example

<p:dummyText size="3"/>

This generates Lorem Ipsum content, for a total of 3 paragraphs.

Attributes

NamePurpose
sizeOptional - The number of paragraphs to generate.
